# Supabase RLS 정책 도우미

## 절차
1. 권한 표부터 만든다: 행=역할(익명/authenticated), 열=CRUD, 셀=✅/❌
2. 표의 ✅ 하나당 정책 1개를 만든다 (정책 이름은 "역할_동작" 형식)
3. 정책을 만들 때마다 반대 역할로 접근해 차단되는지 실험한다

## 규칙
- 정책이 없는 동작은 자동 차단됨을 항상 먼저 확인한다
- service_role key로 우회하는 해결책은 제안하지 않는다
- 개인정보 컬럼이 있으면 select 허용 범위를 반드시 되묻는다
