## 프로젝트 개요
- 주말 독서 토론 참가 관리 서비스 (고객 페이지 + 관리자 화면)
- 파일 구성: index.html(고객), admin.html(관리자) — 각각 단일 파일

## 기술 규칙 (항상 지킬 것)
- HTML/CSS/JS 단일 파일 구성, 프레임워크 사용 금지
- Supabase JS 라이브러리는 CDN 방식으로 불러온다
- 테이블: reservations (name, contact, hope_date, request, status[기본값 "대기"], created_at)
- service_role key와 비밀번호는 절대 코드에 넣지 않는다

## 범위 규칙 (절대 만들지 말 것)
- 고객 회원가입 X / 관리자 가입 기능 X / 결제 X
